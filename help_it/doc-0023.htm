File ENI

Il file ENI (EtherCAT Network Information) è il file di configurazione utilizzato nelle reti EtherCAT per descrivere la topologia di rete e i dispositivi connessi. Il file ENI è solitamente in formato XML e viene generato o modificato utilizzando strumenti di configurazione come il software di configurazione EtherCAT (ad esempio, EtherCAT Configurator o TwinCAT di Beckhoff).

Attualmente Ecang estrae solo queste informazioni dal file ENI:
Dalla sezione <config/slave/info>, vengono letti il ​​nome e l'indirizzo di ogni slave
Dalla sezione <ProcessImage/Inputs>, vengono letti il ​​nome, la dimensione e l'offset di ogni PDO di input
Dalla sezione <ProcessImage/Output>, vengono letti il ​​nome, la dimensione e l'offset di ogni PDO di output

Durante la lettura del file ENI, nella finestra principale viene visualizzato un report delle informazioni estratte.

Attualmente ecang gestisce pdo di dimensione 64, 32, 16 e 8 bit.
Le dimensioni pdo inferiori a 8 bit sono supportate in gruppi di 8 bit. Le dimensioni pdo superiori a 64 bit non sono attualmente supportate.
