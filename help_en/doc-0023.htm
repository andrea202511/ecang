ENI file

The ENI (EtherCAT Network Information) file is the configuration file used in EtherCAT networks to describe the network topology and connected devices. The ENI file is usually in XML format, and is generated or modified using configuration tools such as EtherCAT configuration software (for example, EtherCAT Configurator or TwinCAT from Beckhoff).

Currently Ecang extracts only this information from the ENI file:
From the <config/slave/info> section, the name and address of each slave are read
From the <ProcessImage/Inputs> section the name, size and offset of each input PDO are read
From the <ProcessImage/Output> section the name, size and offset of each output PDO are read

While reading the ENI file a report of the extracted information is displayed in the main window.
