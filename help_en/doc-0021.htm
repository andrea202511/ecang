Option dialog

Language


Allows you to select the language used from those proposed.
The selected language will be active the next time ecang is started

PDO format


It is possible to select the representation format of the PDOs in the output file depending on their length in bits

Tips:
Registers (REG) and the mailbox communication (SDO) always use a RAW representation (first byte read, first byte printed) which cannot be changed in this section
